# Pitch — FinanTec

## Problema

Estudantes e pessoas em início de carreira frequentemente começam a administrar bolsa-estágio, freelas ou o primeiro salário sem uma visão clara sobre receitas, despesas e metas.

Sem organização, pode ser difícil identificar gastos excessivos, criar uma reserva para imprevistos ou planejar compras importantes sem recorrer a dívidas.

---

## Solução

O FinanTec é um assistente de organização financeira com IA generativa voltado para estudantes e pessoas em início de carreira.

Ele utiliza dados financeiros simulados para:

- resumir receitas, despesas e saldo mensal;
- identificar categorias com maior impacto no orçamento;
- explicar conceitos financeiros básicos;
- simular metas financeiras;
- orientar próximos passos simples;
- informar quando não há dados suficientes para responder com segurança.

---

## Diferencial

O FinanTec não funciona apenas como um chatbot genérico.

A aplicação combina:

- análise de dados em Python;
- arquivos CSV e JSON como base de conhecimento;
- cálculos determinísticos para valores financeiros;
- IA generativa para interpretar perguntas e explicar resultados;
- regras para reduzir respostas inventadas;
- limites claros sobre recomendações financeiras.

---

## Público-Alvo

O projeto foi pensado para:

- estudantes universitários;
- estagiários;
- jovens aprendizes;
- pessoas em início de carreira;
- pessoas que desejam começar a organizar receitas, despesas e metas.

---

## Tecnologias Utilizadas

- Python
- pandas
- Streamlit
- CSV
- JSON
- IA generativa via API

---

## Demonstração Esperada

Durante a demonstração, a pessoa usuária poderá perguntar, por exemplo:

- "Em qual categoria eu mais gastei este mês?"
- "Qual foi meu saldo mensal?"
- "Quanto preciso guardar por mês para comprar um notebook?"
- "O que é uma reserva para imprevistos?"
- "Qual banco oferece o melhor CDB hoje?"

Para perguntas relacionadas aos dados cadastrados, o FinanTec responde usando os cálculos e informações disponíveis.

Para perguntas que dependem de dados em tempo real ou recomendação personalizada, o agente explica sua limitação de forma clara.

---

## Impacto Esperado

O FinanTec demonstra como IA generativa pode apoiar uma pessoa em tarefas simples de organização financeira, desde que seja combinada com dados estruturados, cálculos confiáveis e limites de segurança.

O objetivo não é substituir profissionais financeiros, mas oferecer uma experiência educativa e acessível para quem está começando a cuidar da própria vida financeira.
